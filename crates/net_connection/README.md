# ohos-net-connection-binding

This crate is a binding for the net connection module in OpenHarmony.

## Install

```shell
cargo add ohos-net-connection-binding
```

## Usage

Add this crate to your native module and import it from Rust code:

```rust
use ohos_net_connection_binding as net_connection;

// Use the safe Rust APIs exposed by `ohos-net-connection-binding` from your native module.
```
