# ohos-sensor-binding

This crate is a binding for the sensor module in OpenHarmony.

## Install

```shell
cargo add ohos-sensor-binding
```

## Usage

Add this crate to your native module and import it from Rust code:

```rust
use ohos_sensor_binding as sensor;

// Use the safe Rust APIs exposed by `ohos-sensor-binding` from your native module.
```
