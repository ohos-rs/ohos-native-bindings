# ohos-init-binding

This crate is a binding for the init module in OpenHarmony.

## Install

```shell
cargo add ohos-init-binding
```

## Usage

Add this crate to your native module and import it from Rust code:

```rust
use ohos_init_binding as init;

// Use the safe Rust APIs exposed by `ohos-init-binding` from your native module.
```
