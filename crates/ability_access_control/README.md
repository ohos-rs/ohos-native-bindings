# ohos-ability-access-control-binding

This crate is a binding for the ability-access-control module in OpenHarmony.

## Install

```shell
cargo add ohos-ability-access-control-binding
```

## Usage

Add this crate to your native module and import it from Rust code:

```rust
use ohos_ability_access_control_binding as ability_access_control;

// Use the safe Rust APIs exposed by `ohos-ability-access-control-binding` from your native module.
```
