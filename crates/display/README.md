# ohos-display-binding

This crate is a binding for the display module in OpenHarmony.

## Install

```shell
cargo add ohos-display-binding
```

## Usage

Add this crate to your native module and import it from Rust code:

```rust
use ohos_display_binding as display;

// Use the safe Rust APIs exposed by `ohos-display-binding` from your native module.
```
