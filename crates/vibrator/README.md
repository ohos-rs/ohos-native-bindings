# ohos-vibrator-binding

This crate is a binding for the vibrator module in OpenHarmony.

## Install

```shell
cargo add ohos-vibrator-binding
```

## Usage

Add this crate to your native module and import it from Rust code:

```rust
use ohos_vibrator_binding as vibrator;

// Use the safe Rust APIs exposed by `ohos-vibrator-binding` from your native module.
```
