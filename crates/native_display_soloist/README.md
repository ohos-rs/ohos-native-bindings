# ohos-display-soloist-binding

This crate is a binding for the NativeDisplaySoloist module in OpenHarmony.

## Install

```shell
cargo add ohos-display-soloist-binding
```

## Usage

Add this crate to your native module and import it from Rust code:

```rust
use ohos_display_soloist_binding as display_soloist;

// Use the safe Rust APIs exposed by `ohos-display-soloist-binding` from your native module.
```
