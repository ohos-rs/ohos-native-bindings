# ohos-net-stack-binding

This crate is a binding for the net stack module in OpenHarmony.

## Install

```shell
cargo add ohos-net-stack-binding
```

## Usage

Add this crate to your native module and import it from Rust code:

```rust
use ohos_net_stack_binding as net_stack;

// Use the safe Rust APIs exposed by `ohos-net-stack-binding` from your native module.
```
