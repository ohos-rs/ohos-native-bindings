# ohos-qos-binding

This crate is a binding for the qos module in OpenHarmony.

## Install

```shell
cargo add ohos-qos-binding
```

## Usage

Add this crate to your native module and import it from Rust code:

```rust
use ohos_qos_binding as qos;

// Use the safe Rust APIs exposed by `ohos-qos-binding` from your native module.
```
