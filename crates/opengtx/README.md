# hms-opengtx-binding

This crate is a binding for OpenGTX in HarmonyOS.

## Install

```shell
cargo add hms-opengtx-binding
```

## Usage

Add this crate to your native module and import it from Rust code:

```rust
use hms_opengtx_binding as opengtx;

// Use the safe Rust APIs exposed by `hms-opengtx-binding` from your native module.
```
