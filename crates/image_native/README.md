# ohos-image-native-binding

This crate is a binding for the image native module in OpenHarmony.

## Install

```shell
cargo add ohos-image-native-binding
```

## Usage

Add this crate to your native module and import it from Rust code:

```rust
use ohos_image_native_binding as image_native;

// Use the safe Rust APIs exposed by `ohos-image-native-binding` from your native module.
```
