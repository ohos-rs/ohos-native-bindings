# ohos-asset-binding

This crate is a binding for the asset module in OpenHarmony.

## Install

```shell
cargo add ohos-asset-binding
```

## Usage

Add this crate to your native module and import it from Rust code:

```rust
use ohos_asset_binding as asset;

// Use the safe Rust APIs exposed by `ohos-asset-binding` from your native module.
```
