# ohos-bundle-binding

This crate is a binding for the bundle module in OpenHarmony.

## Install

```shell
cargo add ohos-bundle-binding
```

## Usage

Add this crate to your native module and import it from Rust code:

```rust
use ohos_bundle_binding as bundle;

// Use the safe Rust APIs exposed by `ohos-bundle-binding` from your native module.
```
