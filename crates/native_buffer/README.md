# ohos-native-buffer-binding

This crate is a binding for the native buffer module in OpenHarmony.

## Install

```shell
cargo add ohos-native-buffer-binding
```

## Usage

Add this crate to your native module and import it from Rust code:

```rust
use ohos_native_buffer_binding as native_buffer;

// Use the safe Rust APIs exposed by `ohos-native-buffer-binding` from your native module.
```
