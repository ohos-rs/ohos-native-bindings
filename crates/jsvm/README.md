# ohos-jsvm-binding

This crate is a binding for the JSVM module in OpenHarmony.

## Install

```shell
cargo add ohos-jsvm-binding
```

## Usage

Add this crate to your native module and import it from Rust code:

```rust
use ohos_jsvm_binding as jsvm;

// Use the safe Rust APIs exposed by `ohos-jsvm-binding` from your native module.
```
